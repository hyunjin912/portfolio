Cheetsheet

  • 새 파일 만들기 : Ctrl + N
  • 레이어 패널 : F7
  • 작업 취소하기 : Ctrl + Z
  • 이미지 불러오기 : Ctrl + O
  • 100% 비율로 조정하기 : Ctrl + 1
  • 화면 확대하기 : Ctrl + +
  • 화면 축소하기 : Ctrl + -
  • 환경설정 : Ctrl + K
  • 크기 변경하기 : Ctrl + T
  • 전경색으로 채우기 : Alt + DEL
  • 배경색으로 채우기 : Ctrl + DEL
  • 줄 자 : Ctrl + R
  • 레이어 복사 : Ctrl + J
  • 영역 해제 : Ctrl + D
  • Hue/Saturation : Ctrl + U
  • 다수의 레이어 합치기 : Ctrl + E
    (글씨를 이미지화할 때 자주 사용)
  • 영역 반전 : Shift + Ctrl + I
